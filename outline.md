# TIREKCARE - Структура проекта

## Файлы проекта
- `index.html` - Главная страница с блоком отзывов
- `specialists.html` - Страница специалистов и достижений
- `auth.html` - Страница входа/регистрации
- `catalog.html` - Каталог видеоматериалов
- `profile.html` - Личный профиль пользователя
- `player.html` - Видеоплеер
- `support.html` - Страница поддержки и помощь
- `main.js` - Основной JavaScript файл
- `firebase-config.js` - Настройки Firebase

## Навигация
- Главная → index.html
- Специалисты → specialists.html
- Каталог → catalog.html
- Профиль → profile.html
- Поддержка → support.html
- Вход/Регистрация → auth.html

## Особенности
- Отзывы на главной странице в горизонтальной карусели
- Специалисты в виде сетки с фото и достижениями
- Чат поддержки на всех страницах
- Адаптивный дизайн для планшетов и смартфонов
